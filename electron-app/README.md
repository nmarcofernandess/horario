# EscalaFlow

App Electron + React para gestão de escala de trabalho.

## Comandos

```bash
# Rodar no navegador (API + frontend) — ideal para dev
npm run online

# Rodar no Electron (desktop)
npm run electron
```

## Scripts auxiliares

- `npm run api` — inicia só a API Python (porta 8000)
- `npm run dev:web` — inicia só o frontend Vite (porta 5173)

## Requisitos

- Python 3 com `uvicorn` e dependências do projeto
- Node.js 18+
